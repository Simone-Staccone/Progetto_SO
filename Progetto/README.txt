PER COMPILARE IL PROGETTO USARE IL MAKEFILE (COMPILARE SEMPLICEMENTE CON "make"),OPPURE COMPILARE SINGOLARMENTE "server.c -o server" e 
"client.c -o client".PER POI LANCIARE IL SERVER CON ./server 7000 DOVE 7000 RAPPRESENTA IL NUMERO DI PORTA.
IL CLIENT INVECE VA LANCIATO COME ./client 0 7000, DOVE 0 RAPPRESENTE L'INDIRIZZO IP DEL SERVER E 7000 DEVE ESSERE LO STESSO NUMERO
DI PORTA DEL SERVER.
IMPORATNE!!!!!!!
IL FILE "local.c" DEVE STARE NELLA STESSA DIRECTORY DI "server.c" e di "client.c".
